# aoc25
Advent of code 2025 in cpp. In files are read using `fileRead.cpp`.

# Setup
Run `SetupDay.sh` or `SetupDay.bat` with the day number as argument. It will setup the necessary folders, cpp files, and in files.

Run `CompileScripts.sh` or `CompileScripts.bat` with the day number as argument. It will find the cpp file, compile it, and run it with the corresponding in file.
